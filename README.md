## ğŸš€ Getting Started

### Prerequisites
- Flutter SDK 3.13+  
- Android Studio / Xcode (for iOS build)  
- Git  

### Installation (for development)
1. Clone the repository:
   ```bash
   git clone https://github.com/Rexaintreal/Apollo.git
   cd Apollo
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Build for your platform:
   ```bash
   flutter run
   ```
